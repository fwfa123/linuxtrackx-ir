Welcome to Linuxtrack X-IR!

First of all, thank you for using Linuxtrack X-IR, a modern fork of the original Linuxtrack head-tracking software for Linux and Windows.

Linuxtrack X-IR allows you to use a web-camera, wiimote or any of the supported NP devices (currently TrackIR2 to 5, SmartNav3 and SmartNav4) as a head-tracking device, facilitating view control in simulators (flight, racing) or any other type of software.

Among applications utilizing Linuxtrack X-IR are for example X-Plane flight simulator, Flight Gear flight simulator, and any TrackIR enabled game running in Wine (Microsoft Flight Simulator, DCS World, IL-2 Sturmovik, War Thunder, Elite Dangerous, Euro Truck Simulator 2, American Truck Simulator, ...).

What's New in Linuxtrack X-IR:

  • Qt5 Modernization: Updated from Qt4 to Qt5 for modern Linux distributions
  • Wine Compatibility: Enhanced Wine integration using winegcc for Windows games
  • Modern Build System: Updated autotools and CMake support
  • Enhanced Security: PIE, stack protector, and fortify source enabled by default
  • Active Development: Regular updates and community support

Should you encounter a bug, please take a few minutes and file a bug report at the project page - it will allow us to fix the bug that is bothering you, and it also helps to improve Linuxtrack X-IR itself. Also any comments, ideas for improvement and constructive criticism is very welcome!

Note: This is a fork of the original Linuxtrack project by uglyDwarf. The original project has slowed development, and this fork continues with modern build systems and enhanced compatibility while maintaining full attribution to the original authors.
