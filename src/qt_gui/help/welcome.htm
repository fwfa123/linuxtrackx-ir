Welcome to Linuxtrack X-IR v0.99.22!

First of all, thank you for using Linuxtrack X-IR v0.99.22, a head-tracking software specifically tailored for Linux and supporting TrackIR devices.

Linuxtrack X-IR allows you to use your TrackIR device as a head-tracking device, facilitating view control in simulators (flight, racing) or any other type of software.

Among applications utilizing Linuxtrack X-IR are for example X-Plane flight simulator, Flight Gear flight simulator, and any TrackIR enabled game running in Wine (Condor soaring simulator, Il-2, Falcon 4, Tom Clancy's H.A.W.X, ARMA GOLD, ARMA 2).

Note: While the legacy code to support web-camera, Wiimote, and Mac OS is still present within the application, it may not be fully functional or tested. Future updates might address these aspects, but currently, my focus is on ensuring full compatibility with Linux and TrackIR devices.

Special thanks to UglyDwarf for creating Linuxtrack and making this incredible tool available. I am now maintaining LiinuxtrackX-IR with the goal of keeping it functional and up-to-date on modern Linux operating systems.

If you want to know more, see the following pages:

  • X-Plane.org Forums, namely its Linux forum
  • FreeTrack project page - the "model" creation hints, tutorials and more
