Miscellaneous

This pane contains various settings and information that don't fit into other categories.

Logging

LinuxTrack can log various events to help with debugging. The log level determines how much information is recorded:

None: No logging (fastest performance)

Error: Only error messages

Warning: Warnings and errors

Info: General information, warnings, and errors

Debug: Detailed debugging information (slowest performance)

Higher log levels provide more detailed information but may impact performance. For normal use, "Info" or "Warning" is recommended.

Profile Management

Profiles allow you to save and load different configurations for different games or setups. Use the profile management buttons to:

  • Save: Save current settings to a profile
  • Load: Load settings from a saved profile
  • Delete: Remove an existing profile

Reset Options

The reset buttons allow you to quickly restore default settings:

  • Reset All: Restore all settings to their default values
  • Reset Current: Reset only the current pane's settings

Note: Resetting will discard any unsaved changes. Make sure to save your current configuration if needed.
